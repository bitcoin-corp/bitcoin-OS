Bitcoin Email

Decentralized Email on the BSV Blockchain

Send emails with micropayments to eliminate spam. Store message hashes on-chain for immutability. Integrate with HandCash for seamless BSV payments.

View on GitHub Get Started Live Demo

Key Features

💰

Micropayments

Send emails with BSV micropayments attached. Pay fractions of a cent per character to ensure your message gets through while eliminating spam.

🔗

On-Chain Storage

Store email hashes on the BSV blockchain for permanent, tamper-proof records. Verify message authenticity and timestamp with blockchain proof.

🔒

End-to-End Encryption

All emails are encrypted before transmission. Only the intended recipient can decrypt and read the message content.

💳

HandCash Integration

Seamlessly connect your HandCash wallet to send and receive payments. Use $handles for easy addressing and instant BSV transfers.

📮

Digital Stamps

Attach collectible digital stamps to your emails. Premium stamps increase delivery priority and add a personal touch.

🌐

Decentralized

No central server controls your emails. Messages are distributed across the BSV network for true censorship resistance.

Getting Started

Prerequisites

  • Node.js 18+ and npm/yarn
  • HandCash Developer Account
  • Google Cloud Console Account (for OAuth)
  • BSV Wallet with small balance for testing

Installation

# Clone the repository
git clone https://github.com/bitcoin-apps-suite/bitcoin-email.git
cd bitcoin-email

# Install dependencies
npm install

# Copy environment variables
cp .env.example .env.local

# Configure your credentials in .env.local
# Add your HandCash App ID, Secret, and Google OAuth credentials

# Run the development server
npm run dev

# Open http://localhost:3000

Configuration

Set up your environment variables in `.env.local`: 

# HandCash OAuth
NEXT_PUBLIC_HANDCASH_APP_ID=your_app_id
HANDCASH_APP_SECRET=your_app_secret

# Google OAuth
GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_client_secret

# Email Service
EMAIL_FROM=noreply@yourdomain.com
EMAIL_PASSWORD=your_app_password

# BSV Network
BSV_NETWORK=mainnet # or testnet
BSV_PRIVATE_KEY=your_private_key

API Documentation

POST /api/email/send

Send an email with BSV micropayment

{
  "to": ["recipient@example.com"],
  "subject": "Hello World",
  "html": "
Email content
",
  "text": "Plain text content",
  "paymentAmount": 0.00001,
  "encryptEmail": true,
  "storeOnChain": false
}
GET /api/email/receive

Retrieve emails from connected accounts

Query Parameters:
?limit=20&offset=0&folder=inbox
POST /api/auth/handcash

Authenticate with HandCash OAuth

{
  "authToken": "handcash_auth_token"
}
POST /api/blockchain/store

Store email hash on BSV blockchain

{
  "emailHash": "sha256_hash_of_email",
  "timestamp": 1234567890,
  "sender": "$handle",
  "recipient": "user@example.com"
}
