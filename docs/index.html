Chrome Extension Installation

  1. Download the extension ZIP file
  2. Unzip the downloaded file to a folder
  3. Open Chrome and navigate to chrome://extensions/
  4. Enable "Developer mode" in the top right
  5. Click "Load unpacked" and select the unzipped folder
  6. The Bitcoin OS icon will appear in your extension bar

macOS App Installation

  1. Download the DMG file
  2. Double-click to open the DMG
  3. Drag Bitcoin OS to your Applications folder
  4. Launch Bitcoin OS from Applications
  5. Right-click the dock icon and select "Keep in Dock"

Build from Source

  1. Clone the repository: git clone https://github.com/bitcoin-corp/bitcoin-OS.git
  2. Install dependencies: npm install
  3. For Chrome: cd chrome-app && node build.js
  4. For macOS: cd macos-app && npm run dist
  5. Find built apps in the respective dist folders
